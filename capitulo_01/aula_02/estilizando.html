Para Casa

Crie um documento e siga as seguintes instruções

  1. Crie um parágrafo comum
  2. Crie um parágrafo com a classe "destaque"
  3. Crie um parágrafo com o ID "importante"
  4. Estilize os parágrafos com as seguintes regras:
    • Parágrafo comum: cor branca, tamanho 1em, fonte Arial
    • Parágrafo com a classe "destaque": cor vermelha, tamanho 1.2em, fonte Arial, negrito
    • Parágrafo com o ID "importante": cor amarela, tamanho 0.8em, fonte Arial
Para o para casa pode abrir a tag style dentro do documento html
